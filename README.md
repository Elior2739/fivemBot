# FiveM Status Bot
## Setup

Requirements: Lastest version of node.js

1. Open Command Prompt on the folder that the bot in on.
2. Type: `npm install`.
3. Go to config.json and place there the bot's token + server id.
4. Open Command Prompt again and type `node .`.
